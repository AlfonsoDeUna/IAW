# Ejercicios de evaluación de formularios Web


### Ejercicio 1: Crea un formulario web que pida u nombre de usuario y valide que no esté vacío. Si hay error debes modificar el estilo para que el cuadro de texto esté en rojo y puedas añadir algún tipo de pista


### Ejercicio 2: Crea un formulario web que pida un número del 1 al 10 que no admita texto y que no esté vacío. Si hay error debes modificar el estilo para que el cuadro de texto esté en rojo y puedas añadir algún tipo de pista


### Ejercicio 3: Crea un formulario que recoja una contraseña y que tenga una serie de reglas 7 caracteres, una Mayúscula y un caracter extraño.
